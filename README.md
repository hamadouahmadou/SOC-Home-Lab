# SOC Home Lab

## Overview

This project documents the design and implementation of my Security Operation Center (SOC) Home Lab.

The purpose of this project is to gain practical Blue Team and SOC analyst experience by building an enterprise-like environment capable of collecting, detecting and investigating security events.

This lab will continue growing throughout my cybersecurity studies and will eventually include threat hunting, incident response, malware analysis and attack simulations.

---

# Objectives

-Built a professional SOC Home Lab
-Learn linux administration
-Deply wazuh SIEM
-Monitor windows and linux logs
-Simulate cyber attacks
-Investigate security alerts
-Practice incidence response
